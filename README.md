# SIMULACIONES_V1

Proyecto personal desarrollado durante mi incorporación a **Brighter Perú S.A.C.** como practicante de desarrollo de software. Contiene dos simulaciones web interactivas construidas con HTML, CSS y JavaScript vanilla.

---

## 🏢 Simulación — Brighter Perú

Recreación interactiva de la plataforma web de Brighter: flujo de compra, configurador de productos, Academy y sistema de demos.

**Páginas:**
- `index.html` — Landing principal con productos destacados y modalidades de venta
- `tienda.html` — Catálogo con filtros, configurador de tamaños/modelos y add-ons
- `academy.html` — Brighter Academy con cursos LMS y ERP Odoo
- `demo.html` — Sistema de solicitud de demostración (modalidad B2B)
- `carrito.html` — Carrito + checkout con generación de orden

**Tecnologías:** HTML5 · CSS3 (custom properties) · JavaScript ES6 · localStorage

---

## 🚀 Aprende a Programar

Página educativa interactiva para aprender los fundamentos de Kotlin y Android Studio desde cero, con animaciones y gamificación.

**Temas:**
- POO: Clase, Objeto, Constructor, Atributos, Métodos
- 4 Pilares: Encapsulamiento, Herencia, Polimorfismo, Abstracción
- Bases de Datos: CRUD interactivo, JOIN animado, SQL vs NoSQL
- Android Studio: data class, componentes, estructura de proyecto
- Quiz de 10 preguntas con puntaje

**Archivo:** `aprende-programacion.html`
